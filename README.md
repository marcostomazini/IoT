IoT
===

Hackathon IoT - Telefonica Vivo

## Description
Aplicativo Para Monitoramento do Bem Estar Comum

## Features
- Painel de Acompanhamento de Umidade Relativa do Ar, Temperatura, Luminosidade.
- Umidade Relativa: Quando a Mesma Alcança Um Determinado X de Valor Configurado no Painel de Configurações o Serviço Dispara Um 
Comando 'GET' para o Raspberry PI Que Também Está Ligado em Rede e Que Liga o Umidificador de Ar Contribuindo Com a Saúde. (Não Implementado)
- Temperatura: Quando a Mesma Alcança Um Determinado X de Valor Configurado no Painel de Configurações o Serviço Dispara Um 
Comando 'GET' para o Raspberry PI Que Também Está Ligado em Rede e Que Liga ou Desliga o Ar Condicionado. (Não Implementado)
